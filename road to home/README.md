# Тропинка домой 🌴

Тропинка домой: VK Mini App игра для малышей

Мини-игра для детей ~2 лет в формате VK Mini App. Зверята появляются внизу и ждут своей очереди, а мир вокруг них медленно едет вертикально (сверху вниз, туда-сюда), по очереди открывая озеро, сосновый лес, пальмовый остров, пустыню и цветочную поляну. Нужно дождаться, когда появится нужный домик, и перетащить к нему подходящего зверя — рыбку в озеро, лису в лес, льва на остров, бабочку на поляну. За каждое верное совпадение — +1 к счёту.

## Возможности

- Плавная смена дня/ночи фона по реальному времени суток
- Движущийся вертикальный мир (параллакс + прокрутка локаций)
- Перетаскивание зверят мышью/пальцем, у каждого свой звук (Web Audio, без файлов)
- Ввод с клавиатуры: любая буква или цифра выскакивает на экране и озвучивается по-русски (Web Speech API)
- Полноэкранный режим одной кнопкой
- Левая и правая кнопка мыши работают одинаково, системные сочетания клавиш заблокированы
- Оформление в стиле VK Mini App (шапка, счёт, таб-бар)
- Один файл `index.html`, без сборки и зависимостей — только CDN `vk-bridge`

## Технологии

Чистый HTML/CSS/JS, SVG, Web Audio API, Web Speech API (`speechSynthesis`), `@vkontakte/vk-bridge` (только вызов `VKWebAppInit`).

## Как разместить как VK Mini App

1. Включить GitHub Pages для этого репозитория: **Settings → Pages → Source: `main` / `root`** — после сохранения появится ссылка вида:
   `https://<твой-логин>.github.io/games/`
2. Зарегистрировать мини-приложение: [vk.com/apps?act=manage](https://vk.com/apps?act=manage) → «Создать приложение» → тип **«Мини-приложение»**.
3. В настройках приложения указать полученную ссылку на GitHub Pages в поле URL сайта и включить переключатель **«Размещение»**.
4. Приложение станет доступно по адресу `vk.com/app<ID_приложения>`.

Модерация и попадание в общий каталог сервисов VK — отдельный шаг, подаётся через кабинет разработчика (проверка занимает от 3 дней).

## Локальный запуск

Просто открыть `index.html` в браузере — игра работает полностью автономно.
